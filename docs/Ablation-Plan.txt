# ── PHẦN A: 3-Architecture comparison (assignment head only) ────────────────
  Composite function của head IRS-select cho từng arch:

   (1) AE-VQC-PostNN (CURRENT, baseline cho A1):
       f₁ = PostNN ∘ ⟨H⟩ ∘ VQC ∘ E_AE          state(d) → z(n_lat) → ⟨H⟩(|H|) → MLP → logit(|A|)
       Params Case 2: ~55K (PostNN dominate ~50K, AE ~5K, VQC ~96)
       Q:C ratio = 1:500  (classical dominates)
       Attribution: LOW — PostNN có thể replicate bất kỳ quantum mapping nào

   (2) AE-VQC-SoftmaxPQC (proposed, A6 chính):
       f₂ = softmax_β ∘ W·⟨H⟩ ∘ VQC ∘ E_AE     state → z → ⟨H⟩ → linear → softmax_β
       Params Case 2: ~5.8K (AE ~5K, VQC ~96, W |A|×|H|=750, β=1)
       Q:C ratio = 1:8  (classical = thin linear)
       Attribution: MEDIUM-HIGH — linear bound ép quantum làm việc thật

   (3) VQC-SoftmaxPQC pure (Jerbi et al. setup):
       f₃ = softmax_β ∘ W·⟨H⟩ ∘ VQC_re-upload(s)   state → data re-up VQC → ⟨H⟩ → linear → softmax_β
       Params Case 2: ~1K (VQC depth↑ ~250, W 750)
       Q:C ratio = 1:3
       Attribution: HIGHEST — no learned compression, quantum carries all
       ⚠ Underfit risk Case 3: 12 qubits qua 108-dim state chỉ với data re-uploading

  Theory anchor: cả 3 universal approximator (Jerbi App. C.2 + Goto et al. 2021 ref [59]),
  khác biệt là EFFICIENCY (params + samples) đạt universality. (2) là sweet spot cho
  high-dim physical problem (cần AE bridge) + clean attribution.

  Inductive bias key insight (Jerbi App. E.2 Fig 8):
   • DNN: piece-wise linear functions
   • PQC: oscillating Fourier-like sums
   • Wireless physics: oscillating (|Σ e^{jφ}|² beamforming, multipath, discrete phase)
   → (2), (3) MATCH bias với physics; (1) MISMATCH (PostNN piece-wise linear dominates).
   → Theory argument cho paper: (2)/(3) cần ÍT SAMPLE HƠN cho wireless task. (Verify empirical.)

# ── PHẦN D: VQC placement justification (single VQC at top) ─────────────────
  ⚠ Reviewer sẽ hỏi: tại sao VQC CHỈ ở IRS-select, không ở phase/power/Ck?

  Honest justification (paper Section III nên ghi rõ):
   • IRS-select: action space (M+1)^K, per-user UNIQUE, KHÔNG có shared structure
     → classical cần model joint distribution → combinatorial blowup
   • Phase: action space 4^(M·N) NHƯNG PhaseMLP per-element factorization +
     shared weights across N=24 elements → effective complexity N·log(4) tractable
   • Power: 3-axis factored (split·common·private), softmax slots ≤ K+M+1=13 → tractable
   • Common-rate: within-group softmax, group sizes nhỏ → tractable

  → Argument: "VQC at the head where classical sharing/factorization CANNOT reduce
     combinatorial complexity. Other heads have natural classical structure exploitable."

  A/B test ablation cho paper credibility: place VQC ở phase head, đo:
   - Performance gain (likely marginal)
   - Compute cost 4× (4 quantum forwards per step)
   - Param count ~+hundreds (VQC#2)
  Predicted: marginal gain + significant cost → "negative ablation" justify single-VQC.
  CHẠY MỘT LẦN sau A1 chính → đưa vào paper Appendix as ablation supporting design choice.
